Account Switching Test Cases

These tests verify the account switching functionality that allows users to switch between accounts using the 'a' key.

The functionality integrates with fzf for account selection and includes account caching for performance.

Verify accounts are cached and not re-fetched on subsequent calls

Initial Setup

Account: Assets:Checking
Target Amount: $1000.00

  • Assets:Checking
  • Assets:Savings
  • Expenses:Food

Target Account

Assets:Savings

Account Selector Returns

Assets:Savings

Expected Result

✓ Account Changed
